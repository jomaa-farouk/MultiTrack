Installation de MongoDB + Installation de NodeJS 

1/ Lancer un premier terminal , taper mongod
2/ Lancer un deuxième terminal, taper mongo 
3/ Lancer un troisième terminal, accéder au source de app.js (le répertoire du serveur) ( par la commande cd CHEMIN)
4/ Taper npm install (package.json) pour installer les différents plugins (les dépendences)
5/ Taper node app.js
6/ Tester avec les URL existantes dans les différents routes (http://localhost:8080/ ou http://localhost:8080/users)
