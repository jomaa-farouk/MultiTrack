Installation de MongoDB + Installation de NodeJS 

1/ Lancer un premier terminal , taper mongod
2/ Lancer un deuxième terminal, taper mongo 
3/ Lancer un troisième terminal, accéder au dossier "server" (le répertoire du serveur) ( par la commande cd CHEMIN)

4/ Taper "npm install" (package.json) pour installer les différents plugins (les dépendences)
5/ Taper "npm install -g mocha" afin d'installer tout les dépendences de mocha
6/ Taper "npm install mongoose-auto-increment" 

7/ Taper node app.js

8/ Lancer un quatrième terminal, accéder au source de test.js (dossier "server") 
9/ Taper "mocha test.js" pour voir les résultats des différents tests unitaires 
 

10/ Tester si vous voulez avec les URL existantes dans les différents routes (http://localhost:8080/ ou http://localhost:8080/users ou etc... )
